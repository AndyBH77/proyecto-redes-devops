# Proyecto Redes DevOps

Aplicación web hecha con Flask para el proyecto de Redes.

## Ejecutar el programa desde el power shell

```bash
az container delete `
--resource-group rg-redes-devops `
  --name flask-aci `
  --yes
```

Nota: Se tiene que estar en con la sesión iniciada en Microsoft azure para que funcione la app. Gracias.
